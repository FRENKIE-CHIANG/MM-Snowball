# MM-Snowball

MM-Snowball is a multimodal dialogue dataset for studying hallucination accumulation and correction across multi-turn vision-language interactions.

## Dataset

This repository contains:

- `MM-Snowball.json`: dataset annotations.
- `images/`: image files referenced by the dataset.
- `docs/`: static project homepage for GitHub Pages.

Current local release statistics:

- 4,992 annotated image-dialogue samples.
- 6 dialogue turns per sample.
- 5,000 image files.

## Project Page

The project page source is in `docs/index.html`.

After pushing this repository to GitHub, enable it from:

`Settings` -> `Pages` -> `Build and deployment` -> `Deploy from a branch`

Then choose:

- Branch: `main`
- Folder: `/docs`

The page will be published at:

`https://frenkie-chiang.github.io/MM-Snowball/`
